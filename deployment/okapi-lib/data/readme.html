Overview

The Okapi Library is a collection of Java classes design to help in programming components and applications related to translation and localization tasks. It offers a low-level resource model, as well as a set of common API to extract text parts from their original format, modify them, and merge them back. The library also provides facilities to manipulate, annotated and in general, work with the extracted text.

For a list of changes since the previous release, please see the Changes Log document.

Note that the framework is still in its early stage of development. Some of the classes, interfaces and internal workings presented here may change in the upcoming versions of the Okapi Library.

This distribution contains the following:

lib All the files needed to run the Okapi classes.
help The help files that go with the several UI components provided with the library.
config Default configuration files (e.g. segmentation rules, filters parameters, etc.)
examples A few examples on how to use the library, in Java and Python.
localweb/devguide The local copy of the Developer's Guide page for this release.
localweb/javadoc The local copy of the JavaDoc pages for this release.

The latest version of these documents and of the library are always available on-line, at the Okapi Framework Web site.

Windows Platform

Requirements

In order to use the Okapi libraries on Windows you must have a running installation of Java 1.5 or later.

Installation

To install the Okapi libraries on Windows:

  1. Download the ZIP file of the Windows distribution.
  2. Unzip the file on any location, for example: C:\OkapiLib

Uninstallation

To uninstall the Okapi libraries

  1. The Okapi libraries are independently of Windows, so to uninstall it you can simply delete the folder where you have installed them. There is no registry dependencies to remove.

Linux Platforms

Requirements

In order to use the Okapi libraries on Linux you must have a running installation of Java 1.5 or later.

Installation

To install the Okapi libraries on Linux (32- and 64-bit):

  1. Download the ZIP file of the Linux distribution you need (there is one for 32-bit and one for 64-bit).
  2. Extract the GunZip file on any location.

Uninstallation

To uninstall the Okapi libraries:

  1. Delete the directory where the Okapi libraries are installed.

Macintosh Platform

Requirements

On both Mac OS X 10.5 (Leopard) or higher, and Mac OS X 10.4 (Tiger), the Okapi libraries requires Java 1.4.2 or Java 1.5.x. Note that Java 1.6 (which is 64-bit) is not supported yet. Currently the Okapi libraries use Carbon, not Cocoa. Support for Cocoa will come when the Cocoa implementation of SWT is stable.

Installation

To install the Okapi libraries on Macintosh:

  1. Download the ZIP file of the Macintosh distribution.
  2. Extract the file on any location.

Uninstallation

To uninstall the Okapi libraries:

  1. Delete the directory where the Okapi libraries are installed.

Support

The examples directory contains source code of examples in Java and Python that use the Okapi libraries.

If you have questions, suggestions or bug to report, you can do this different ways:

  • Read and post notes in the Okapi Tools users group (you need to be registered member to post, but not to read).
  • Read an post issue report on the Google Code project site.
