Overview

This package provides an adapter to use several of the Okapi filters directly from the computer-assisted translation tool OmegaT. It includes the following filters:

  • The TS Filter -- for Qt TS files
  • The TTX Filter -- for Trados TagEditor TTX files
  • The JSON Filter -- for JSON files

The source code for this package is hosted in the Okapi Framework project.

Requirements and Installation

Requirements

  • Windows: You must have a running installation of Java 1.6 or later (Sun Java, or an OpenJDK VM are recommended).
  • Linux: You must have a running installation of Java 1.6 or later (Sun Java, or an OpenJDK VM are recommended).
  • Macintosh: You must have a running installation of Java 1.6 or later; which are available on Mac OS X 10.5 (Leopard) for 64-bit machines, and on Mac OS X 10.6 (Snow Leopard) for 64-bit and 32-bit machines.
  • Important: in all cases you need Java 1.6 or later. OmegaT requires only Java 1.5 or later

Installation

To install the plugin:

  1. Create a plugins directory in the OmegaT main directory, if it does not exist yet.
  2. Copy the jar file of the distribution in the plugins directory.
  3. The next time you run OmegaT the new filters will be available.

Uninstallation

To uninstall the plugin:

  1. Simply delete the jar file from the plugins directory.
  2. The next time you run OmegaT the filters will not be available.
