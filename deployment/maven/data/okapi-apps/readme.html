Overview

The okapi-apps distribution contains:

  • The okapi-lib distribution, a collection of Java classes design to help in programming components and applications related to translation and localization tasks.
  • Several applications built on top of the framework:
    • Tikal is a command-line tool to execute some simple localization-related tasks, like extraction and merge of XLIFF documents.
    • Rainbow is an application that provides a user interface to launch different translation and localization-related utilities.
    • Ratel is a simple editor for segmentation rules in SRX format.

For a list of changes since the previous release, please see the Changes Log document.

Note that parts of this project are still relatively new and still in development. Some of the classes, interfaces and internal workings presented here may change in the upcoming versions of the Okapi Framework.

lib All the files needed to run the Okapi libraries and applications.
help The help files.
config Various default configuration files (e.g. segmentation rules, additional filter configurations, etc.)

The latest version of all documents and help is always available on-line, at the Okapi Framework Web site.

Windows Platform

Requirements

In order to run the Okapi applications on Windows you must have a running installation of Java 1.5 or later.

Installation

To install Rainbow on Windows:

  1. Download the ZIP file of the Windows distribution.
  2. Unzip the file on any location, we recommend C:\Program Files\Rainbow6.

To run Rainbow: simply double-click on the rainbow.exe file in the main folder.

You can also run the application with the following command line (assuming it has been unzipped in the recommended folder):

C:\>javaw -jar C:\Program Files\Rainbow6\lib\rainbow.jar

Uninstallation

To uninstall Rainbow

  1. Rainbow runs independently of Windows, so to uninstall it you can simply delete the folder where you have installed it. There is no Registry information to remove.
  2. If you want you can also remove the file where the user preferences are saved. It is located in the Home folder of the user and called .Rainbow.

Linux Platforms

Requirements

In order to run the Okapi applications on Linux you must have a running installation of Java 1.5 or later.

Installation

To install the Okapi applications on Linux (32- and 64-bit):

  1. Download the ZIP file of the Linux distribution you need (there is one for 32-bit and one for 64-bit).
  2. Extract the GunZip file on any location.
  3. Make sure the execution rights are set for the files called rainbow.sh , ratel.sh, and tikal.sh on the main directory.

To run, for example, Rainbow: simply double-click on the rainbow.sh file in the main directory.

You can also run the application from the command line (assuming Rainbow is installed in \home\archie\Rainbow6):

archie@archie:~>java -jar Rainbow6/lib/rainbow.jar

Uninstallation

To uninstall the Okapi applications:

  1. Delete the directory where the okapi application are installed.
  2. If you want, you can also remove the files where the user preferences are saved. They are located in the Home folder of the user and called .Rainbow and .Ratel.

Macintosh Platform

Requirements

Use the Cocoa distribution for Macintsoh OS X 10.5 (Leopard). Use the Carbon distribution for Macintosh OS X 10.4 (Tiger) or 10.5 (Leopard). Both will use Java 1.4.2 or Java 1.5.x (not Java 1.6, which runs only on 64-bit)

Installation

To install the Okapi application on Macintosh:

  1. Download the ZIP file of the Macintosh distribution.
  2. Extract the file on any location. For example /Users/admin/Desktop/Rainbow.

To run, for example, Rainbow:

  1. Try double-clicking on the rainbow.sh shell script that is in the main directory of the distribution.
  2. If this does not work:
    1. Open a terminal window.
    2. Go to the main directory of the distribution, for example /Users/admin/Desktop/rainbow
    3. Execute the following command:
    java -XstartOnFirstThread -jar lib/rainbow.jar

You MUST specify the -XstartOnFirstThread option or the application may not work properly.

Uninstallation

To uninstall the Okapi applications:

  1. Delete the directory where the okapi application are installed.
  2. If you want, you can also remove the files where the user preferences are saved. They are located in the Home folder of the user and called .Rainbow and .Ratel.
