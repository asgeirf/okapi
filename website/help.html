Support

If you were not able to find an answer to your question in the on-line help:

Try to send feedback or ask question directly by selecting the Send Feedback command in the Help menu of most Okapi applications.

Try the Okapi Tools users group mailing list:

  • See the mailing list archives
  • Subscribe to the user group
  • Post a message

You can also post bug reports and request for features on the Google Code project site (click New Issue).
