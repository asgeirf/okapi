Developers Notes

Developing Tools with the Framework

You can use the Okapi components like a toolkit to write your own programs and scripts:

  • The Getting Started section of the Developer's Guide goes through the basics of the framework and should be enough to get you started in using the components in your own programs or scripts.
  • The Glossary can be a useful reference.

You can also develop your own components and used them with the framework:

  • The other sections of the Developer's Guide goes into deeper details and allow to understand better how the framework's components interact.
  • The Java Documentation of the framework's main shared classes is also a good source of information.

General guidelines are also available:

  • Coding guidelines
  • Help authoring guidelines

Source Code of the Framework

The source code of the Okapi Framework and its applications is freely available from Google Code project's Web site:

  • Subversion repository: http://code.google.com/p/okapi/source/checkout
  • Latest code commits: http://code.google.com/p/okapi/source/list
  • Source tree browsing: http://code.google.com/p/okapi/source/browse

If you want to setup the development environment of Okapi under Eclipse, please, follow the instructions provided on this Wiki page.
