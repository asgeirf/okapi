Latest Releases

All the packages, current and passed, are accessible from the Download area of the Google Code project site
The latest releases are:

===> Rainbow - Release 1, Version 6.0.1_Beta19
Rainbow is a toolbox that provides various utilities to help you in localization tasks. This package includes both the classes that are in the Filters Library as well as in Ratel.

  • Downloads: Windows distribution | Linux distribution | Macintosh distribution
  • Online Help | Release Notes | Screenshots

===> Okapi Library - Release 1, Beta18
The Filters Library is a Java library with all the classes needed to use the Okapi filters components with your own program and scripts.

  • Download: Windows distribution | Linux distribution | Macintosh distribution
  • Developer's Guide: Getting Started | Table of Contents

===> Ratel - Release 1, Version 1.0.1_Beta19
Ratel is a program to create and maintain segmentation rules in the SRX (Segmentation Rule eXchange) format. (Note: the functionality of Ratel is included in the Rainbow distribution)

  • Downloads: Windows distribution | Linux distribution | Macintosh distribution
  • Online Help | Screenshots

Older .NET Releases

Older releases of the .NET implementations of the Okapi Framework , such as Rainbow v5 and Olifant are still available, but are no longer developed.

See this site: http://okapi.sourceforge.net/downloads.html for the downloads of the .NET applications.
