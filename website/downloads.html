Downloads

Latest Releases

All the packages, current and passed, are accessible from the Download area of the Google Code project site
The latest releases are:

>>> Okapi Applications (okapi-apps) - Version @okapiappsVersion@

This distribution is for users and developers. It includes all the files of the okapi-lib distribution (except the developers documentation), as well as the UI components of the framework, and several applications: Rainbow (version @rainbowVersion@), Ratel (version @ratelVersion@), CheckMate (version @checkmateVersion@) and Tikal (version @tikalVersion@).

For Windows 32-bit ==> okapi-apps_win32-x86_@okapiappsVersion@.zip
For Linux 32-bit ==> okapi-apps_gtk2-linux-x86_@okapiappsVersion@.tar.gz
For Linux 64-bit ==> okapi-apps_gtk2-linux-x86_64_@okapiappsVersion@.tar.gz
For Mac OS X 1.6 Intel (Snow Leopard 32-bit) ==> okapi-apps_cocoa-macosx_@okapiappsVersion@.tar.gz
For Mac OS X 1.6 Intel (Snow Leopard 64-bit) or 1.5 (Leopard 64-bit) ==> okapi-apps_cocoa-macosx-x86_64_@okapiappsVersion@.tar.gz

The list of changes between the previous release and the latest is available in the Changes Log.

>>> Okapi Library (okapi-lib) - Version @okapilibVersion@

This distribution is mostly for developers. It includes all the core libraries, filters, and other non-UI components of the Okapi Framework. It also includes Tikal (version @tikalVersion@). You want this distribution if you need only the non-UI parts of the framework.

For all platforms (.zip format) ==> okapi-lib_all-platforms_@okapilibVersion@.zip
For all platforms (.tar.gz format) ==> okapi-lib_all-platforms_@okapilibVersion@.tar.gz

Latest Snapshots

The latest snapshot builds are the latest successful builds created from the source code being currently developed. They have the most up-to-date features, but they may be untested and some parts maybe unstable. They are the versions used by the developers.

The latest snapshots are located here: http://okapi.opentag.com/snapshots The dates and times of the files are in US Pacific Time.

Older .NET Releases

Older releases of the .NET implementations of the Okapi Framework , such as Rainbow v5 and Olifant are still available, but are no longer developed.

See this site: http://okapi.sourceforge.net/downloads.html for the downloads of the .NET applications.
