Latest Releases

All the packages, current and passed, are accessible from the Download area of the Google Code project site
The latest releases are:

===> Okapi Applications (okapi-apps) - Version @okapiappsVersion@
This distribution includes the Okapi Library as well as Rainbow (version @rainbowVersion@), Ratel (version @ratelVersion@) and Tikal (version @tikalVersion@).

  • Downloads: Windows | Linux | Linux 64-bit | Macintosh Cocoa | Macintosh Carbon

===> Okapi Library (okapi-lib) - Version @okapilibVersion@
The Okapi Library is a Java library with all the classes needed to use the Okapi filters components with your own program and scripts. This distribution also include Tikal (version @tikalVersion@).

  • Download:  All Platforms (ZIP) | All Platforms (GZIP)

Latest Snapshots

The latest snapshot builds are the latest successful builds created from the latest source code. They have the most up-to-date features, but they may be untested and unstable. They are the versions used by the developers. Use them at your own risk.

The latest snapshots of all the builds at located here: http://okapi.opentag.com/snapshots The date and time listed are in US Pacific Time.

Older .NET Releases

Older releases of the .NET implementations of the Okapi Framework , such as Rainbow v5 and Olifant are still available, but are no longer developed.

See this site: http://okapi.sourceforge.net/downloads.html for the downloads of the .NET applications.
