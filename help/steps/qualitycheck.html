Overview

This step generate a report of issues found by comparing the source and target of text units.

Takes: Filter events
Sends: Filter events

No check is done on the entries set as non-translatable.

Parameters

Output path -- Enter the full path of the HTML report to generate. You can use the variable ${rootDir} in the path.

Open the result file after completion -- Set this option to automatically open the report file after the process is complete.
