Parameters

Tokenize source -- Set this option to tokenize the content of the source.

Tokenize targets -- Set this option to tokenize the content of the different selected targets.

Select languages, or specify a locale filter -- Specify what locale should be tokenized. Leave the field empty to select all locales.

Select tokens to extract -- Specify what types of tokens should be generated. Leave the field empty to generate all types of token available.
