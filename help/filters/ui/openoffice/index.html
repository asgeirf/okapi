Overview

This filter allows you to process the different types of documents of the OpenOffice.org suite, such as ODT (text documents), ODS (spreadsheets), ODP (presentations) and ODG (drawings) documents.

You can also use the ODF Filter (which comes with the OpenOffice Filter) to process directly OpenDocument XML files if necessary.
