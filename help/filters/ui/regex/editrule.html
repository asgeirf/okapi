Regex Filter Parameters

Edit Rule

This dialog box allows you to edit a rule.

Boundaries of the content

Start -- Enter the regular expression matching just before the content.

End -- Enter the regular expression matching just after the content.

You can enter a example of text to process in the edit field. The result of the rule applied to the example is shown below as you modify the parameters.

For example:

Start: \[.*?\]=
  End: \}

Sample: [ID100]={Text to translate}

Resource name

Before the name -- Enter the regular expression matching just before the name of the resource. This expression is applied on the text matched by the Start expression.

After the name -- Enter the regular expression matching just after the name of the resource. This expression is applied on the text matched by the Start expression.

For example:

Before the name: \[
 After the name: \]

Sample: [ID100]={Text to translate}
