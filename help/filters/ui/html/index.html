Overview

The HTML Filter is an Okapi component that implements the IFilter interface for HTML and XHTML documents. The filter is implement in the class net.sf.okapi.filters.html.HtmlFilter of the Okapi library.

In the current version of the filter the content of <style> and <script> elements is not extracted, and tags from server-side scripts such as PHP, ASPX, JSP, etc. are not formally supported (i.e. some may be fine through, some may not).
