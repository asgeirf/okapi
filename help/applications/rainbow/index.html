Rainbow

Rainbow is a Java application that that offers a palette of utilities aiming to help you during the process of localizing documentation, on-line help and software-related material. The application runs on any platform where SWT is supported, including Windows, Linux and Macintosh.

The utilities Rainbow uses are plug-in components compatible with the Okapi Utility specification. Several are shipped with Rainbow, but you can also create your own and  launch them from Rainbow.

Help and Support

When requesting support, make sure to always specify the version of the tools you are using and what operating system you are running.
