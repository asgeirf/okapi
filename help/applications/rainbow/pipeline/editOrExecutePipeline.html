Edit / Execute Pipeline

This dialog box allows you to create, modify and execute pipelines.

For more detailed information about what pipelines are, see the Pipelines section.

The top text field shows the name of the current pipeline. It is empty if the pipeline is not saved.

The left-hand list is a list of all the steps that make up the current pipeline. The steps are executes from top to botton in the order shown in this list.

The right-hand panel shows the options associated with the step currently selected in the left-hand list. Some steps may not have any option, in that case the panel simply shows the message "This step has no parameters".

Step commands:

Add Step -- Click this button to add a new step in the current pipeline. The Okapi distribution comes with a set of pre-existing steps you can use right out of the box, and you may have additional plugin steps as well.

Remove Step -- Click this button to remove the selected step from the current pipeline.

Move Up -- Click this button to move the selected step upward (the step executed first is the one at the top of the list).

Move Down -- Click this button to move the selected step downward (the step executed last is the one at the bottom of the list).

Pipeline commands:

Load -- Click this button to load an existing pipeline.

New -- Click this button to start a new pipeline.

Save -- Click this button to save the current pipeline. If the current pipeline has no name yet, you will be prompted to provide a file name and location.

Save As -- Click this button to save the current pipeline under a new file name or at a new location.

Below the buttons a text field displays the description of the step currently selected.
