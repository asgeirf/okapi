Versions

Rainbow has been developped since many years. It was first developped as a freeware (from version 1 to version 4). Then, with Rainbow 5, it moved to become an open-source project under the Okapi Framework.

There are two branches currently supported:

Rainbow 6 -- This is the latest application, This application is part of the Okapi Framework on the Java platform which is the most recent iteration of the framework.

Rainbow 5 -- This is a .NET application that is still maintained but where no new developments is done. This application is part of the Okapi Framework developped on the .NET platform.
