Input Document Properties

Filter -- Select the filter to associate with the input document, or choose <None> to associate no filter with the input document.

Parameters -- Select the parameters file associated with the input document, or choose <Defaults> to use the defaults of the selected filter.

Edit -- Click this button to edit the parameters file currently selected. If no parameters specific editor is available, the file is opened in a text editor.

Create --Click this button to create a new parameters file. You will be prompted for its name.

Delete -- Click this button to delete the parameters file currently selected.

Source encoding -- Enter the name of the encoding to use for the source document. Leave this entry empty to use the default source encoding that is defined in the Languages and Encodings tab of the main window.

Target encoding -- Enter the name of the encoding to use for the target document. Leave this entry empty to use the default target encoding that is defined in the Languages and Encodings tab of the main window.
