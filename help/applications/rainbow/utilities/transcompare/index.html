Utility – Translation Comparison

Overview

This utility allows you to compare two sets of translated text.

The input is two document, the first one with the base translation, the second one with the translation to compare to the base translation. The two documents must have the same number of entries and the entries must be in the same order.

For each entry the utility compare the two translated text and calculate a score that indicates how similar the text of the second document is from the text of the first translation. the score is a number between 0 and 100. the score is based on the Levenstein distance and additional processing. You can find more information on that mertic at http://www.merriampark.com/ld.htm and http://en.wikipedia.org/wiki/Levenshtein_distance.

The utility outputs an HTML document for each pair of input document. That file list the pairs of text and the resulting score.

Input Parameters

  • The list of the documents containing the base translation (Input list 1)
  • The list of the corresponding documents.containing the translation to compare to the base one (Input list 2)
  • The source language and default input encoding.
  • The target language and default input encoding.
