Utility – Line-Break Conversion

Overview

This utility allows you to convert the line-breaks of an input document from one type to another.

This utility does not use filters.

The encoding of the document is not modified in any way and Byte-Order-Marks are not added or deleted.

The type of line-break in the input file is detected automatically, and the file may contain a mix of different types of line-breaks.

Common Parameters

  • The list of the documents to modify (Input list 1).
  • The names and locations of the output documents.

Configuration

Options Tab

New type of line-break

DOS/Windows -- Select this option to change all the line-breaks to DOS/Windows line-breaks (CR+LF).

Unix/Linux -- Select this option to change all the line-breaks to Unix/Linux line-breaks (LF).

Macintosh -- Select this option to change all the line-breaks to Macintosh line-breaks (CR).
