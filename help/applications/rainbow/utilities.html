Utilities

Rainbow is distributed with several pre-installed utilities:

  • Translation Package Creation
  • Translation Package Post-Process
  • Text Rewriting
  • ID-Based Alignment
  • Search And Replace
  • Encoding Conversion
  • Line-Break Conversion
  • Byte-Order-Mark Conversion
  • URI Conversion
  • RTF Conversion
  • XSL Transformation
  • Used Characters Listing
  • Translation Comparison
