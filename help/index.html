Okapi Framework - Help

Components Shared Help

The help files in this folder are designed to be used from the user interface of the different Okapi components. Depending on which components you have installed, you may have access to the modules list below. For browsing the complete help system, you can access it on the Okapi Framework Web site.

  • Rainbow (Localization toolbox)
    • The utilities provided with Rainbow
  • Ratel (Segmentation rules editor)
  • The Okapi Filters

See also the online Developer's Guide for more information on how to use the filters in your own scripts and applications, and how to develop new filters.
